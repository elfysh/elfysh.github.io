Политика конфиденциальности (Privacy Policy)

Дата последнего обновления: 29 декабря 2025 года

Настоящая Политика конфиденциальности объясняет, как приложение "Ian8n Workflow App" (далее – "Приложение") собирает, использует и обрабатывает информацию через Pinterest API.

1. Сбор информации

Для обеспечения основных функций Приложения мы будем получать доступ к информации в рамках следующих разрешений Pinterest API:

  • Информация о пинах (pins:read): Доступ к данным, связанным с вашими личными пинами, в первую очередь для получения прямых ссылок для скачивания изображений (URL).

Мы обязуемся запрашивать доступ только к минимально необходимым разрешениям и данным для достижения единственной заявленной цели.

2. Использование информации

Собранная информация (в частности, URL-адреса изображений из пинов) будет использоваться исключительно для следующей единственной цели:

  • Скачивание и личное использование изображений: Приложение с помощью инструмента автоматизации elfysh поможет пользователю скачать строго необходимое минимальное количество изображений с его личных досок Pinterest для использования в личных проектах, создания мудбордов или сбора творческих идей в офлайн-режиме.

3. Обмен и раскрытие информации

Мы ни при каких обстоятельствах не продаем, не обмениваем и не передаем каким-либо иным способом любые данные, полученные через Pinterest API (включая ссылки на изображения), каким-либо третьим лицам.

Весь скачанный контент и доступ к данным ограничены личным компьютером пользователя или его приватной средой n8n и никогда не становятся публичными.

4. Безопасность данных

Мы принимаем разумные технические меры для защиты ваших данных. Все учетные данные API безопасно хранятся в менеджере учетных данных elfysh и используются только при необходимости для отправки запросов к Pinterest.
